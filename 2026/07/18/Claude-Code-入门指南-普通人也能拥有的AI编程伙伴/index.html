上篇：手把手安装配置

🤔 先判断一下：你是否需要 Claude Code？

Claude Code 和网页版 AI 都是同一个 AI 大脑，但使用方式不同。下面这个表格可以帮你判断自己适合哪个：

对比维度 🌐 网页版 AI（DeepSeek、豆包、ChatGPT） 💻 Claude Code
怎么用 打开浏览器就能用 要在终端里打字命令
操作文件 需要手动把文件拖进对话框 直接在你文件夹里工作
一次处理 一次传几个文件 一次看几十个文件
记住你 每次新对话要重新介绍 有长期记忆，自动加载
帮你改文件 你复制粘贴它改的内容 直接修改你电脑上的文件
输出报告 只能生成文字 可生成 Markdown / PDF / PPT / Excel等等格式的文件
适合谁 日常聊天、简单问答 需要 AI 帮你操作电脑文件，处理复杂任务的人

简单一句话判断：

如果你只是想问问题、聊聊天，用 网页版 就好，打开浏览器就能用，没有任何门槛。

如果你想让 AI 帮你整理文件、写文档、处理数据、批量改东西，那就装 Claude Code —— 它能直接进到你的文件夹里工作。

本文接下来的内容，都是针对 Claude Code 的安装和使用。

如果你决定要装 Claude Code，先确认一下你的电脑是否满足系统要求：

最低系统要求：

系统 是否支持
Windows 10 / 11 ✅ 支持
macOS ✅ 支持

为什么 Windows 7 和 8 不行？Claude Code 依赖较新版本的 Node.js（需要 Node.js 18+），而这些版本早已停止支持 Windows 7/8。如果你用的是 Win7/8，建议升级系统到 Windows 10 或 11。

  • 硬盘至少有 500MB 空闲空间

网络要求：

  • 需要能够访问 DeepSeek API（国内网络可以直接访问，无需科学上网）

〇、几个概念

这篇文章里会出现几个容易搞糊涂的词，可以先花两分钟了解一下它们，不需要记住，只要有个大致的印象就行，后面哪里遇到不懂的，再根据左边的目录跳回来。当然如果你已经有过一些相关经验，完全可以跳过。

🖥️ 什么是「终端」「命令行」「cmd」「PowerShell」？

这是最容易搞混的一组概念，这里有一个比喻，可以帮你理清：

电脑系统就像一栋楼。

  • 图形界面（你平时操作的样子） = 楼里的电梯和走廊，点哪里走到哪里，直观
  • 终端 / 命令行 = 楼里的一条密道，你对着它喊一句口令，它直接把你送到指定房间，比走电梯快得多

终端 是一个统称，Windows 上有两个常见终端：

名字 图标 说明
cmd（命令提示符） 黑窗口 + 白文字 老牌终端，Windows 自带，功能基础
PowerShell 蓝窗口或黑窗口 微软新一代终端，功能更强，也是 Windows 自带

Mac 上的终端 就叫「终端」，是一个白底或黑底的窗口。

本文怎么做： 你只需要记住：

  • 终端 = 一个让你打字命令的窗口
  • 文中写 bash xxx 这样的代码块，就表示「把里面的内容复制到终端里，按回车」

🚪 怎么打开终端？

根据你的电脑系统，操作不太一样：

Windows：

方法一（通用，终端默认在用户目录打开）：

  1. 按键盘上的 Win + R（左下角 Windows 键 + R 键）
  2. 屏幕左下角会弹出一个小窗口
  3. 输入 powershell，按回车
  4. 一个深色背景的窗口就打开了——这就是终端

方法二（推荐，直接在指定文件夹打开）：

  1. 打开文件资源管理器，找到你要操作的文件夹
  2. 在文件夹的空白处右键单击
  3. 在弹出的菜单中，选择「在终端中打开」或「Open in Terminal」
  4. 终端直接在这个文件夹里打开了，省去跳转目录的步骤

💡 方法二的好处是终端打开后就在你想要的文件夹里，不用再输 cd 命令跳转。当然你用方法一也完全没问题，你想让claude code操作哪个目录，你就把那个目录的绝对路径复制粘贴到对话框里就行。

Finder 是 Mac 上的文件管理器——相当于 Windows 上的「此电脑」或「文件资源管理器」。你在 Mac 上打开任意一个文件夹看到的就是 Finder 窗口。

显示隐藏文件 查看 → 勾选「隐藏的项目」 Cmd + Shift + .

四、获取 DeepSeek API Key

Claude Code 本身是 Anthropic 公司的产品，但原生支持配置第三方 API。对于国内用户来说，使用 DeepSeek API 是最实惠的方案，价格约为官方 API 的十分之一。

3.1 注册 DeepSeek 账号

  1. 打开浏览器，访问 https://platform.deepseek.com/
  2. 点击右上角「登录」→「注册」
  3. 用手机号或邮箱注册，按提示完成验证
  4. 登录后进入控制台

3.2 创建 API Key

  1. 在左侧菜单点击「API Keys」
  2. 点击「创建 API Key」
  3. 给它起个名字（比如 claude-code），点击创建
  4. ⚠️ 关键：创建成功后立刻复制 Key 并保存到安全的地方！ 这个 Key 只显示一次，关掉页面就再也看不到了。

Key 的格式类似 sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx。

3.3 充值（少量即可）

DeepSeek API 按量计费，充值 10 元就够用很久。注册时通常有免费额度赠送，可以先试试。在左侧菜单「充值」里按提示操作就行。

注意：在下一步配置settings.json前，一定要确保api里是有钱的，哪怕只有一块钱也可以，否则会报错。

下篇：Claude Code 能帮你做什么？

以下案例来自大学生 小 T 的真实使用经历。小 T 是计算机专业大二学生，C 语言基础扎实，正在学数据结构和 STM32 嵌入式开发。

不止写代码——日常办公也能用

很多人听到”命令行工具”就以为这是程序员专属。其实 Claude Code 能做的远不止写代码。它能直接操作你电脑上的文件——读取、修改、创建、转换格式，就像一个住在本地的 AI 助手。

和网页版 AI 最大的区别在哪？ 用网页版（豆包、DeepSeek、ChatGPT 等），你需要把文件一个个拖进对话框，每次对话结束上下文就丢了，下次要重新传。而 Claude Code 直接在你的文件夹里工作——它能一眼看到整个目录里有什么，你不需要手动上传任何东西。改完的文件直接保存回原位置，没有”下载→覆盖→确认”的来回折腾。

说白了一句：网页版 AI 是你把文件送到它面前，Claude Code 是它直接走进你的书房。

举几个普通人立刻能用的例子：

  • 阅读总结：有一份几十页的 PDF 合同或论文？让 Claude Code 帮你读，然后一句话总结要点、标注风险条款
  • 撰写文书：要写一份活动策划、工作汇报、社团申请书？告诉它需求和风格，它直接生成 .md 或 .txt 文件，你稍作修改就能用
  • 处理 Excel / CSV 表格：有一个乱七八糟的数据表？让它帮你整理、筛选、生成统计摘要，几秒钟搞定手工几小时的工作
  • 批量重命名文件：照片、文档、音乐文件命名混乱？让它按规则批量重命名
  • 格式转换：Markdown 转 PDF、JSON 转 CSV、摘取网页内容存为笔记——说一声就行

操作方式很简单，你可以任选一种方式来告诉 Claude Code “我要在这个文件夹里工作”：

方法一：在文件夹里直接打开终端（最推荐）

  1. 打开文件资源管理器，找到你要操作的文件夹
  2. 在文件夹的空白处右键单击
  3. 选择「在终端中打开」或「Open in Terminal」（Windows 11 或 Mac 都有这个选项）
  4. 在弹出来的终端里输入 claude，回车
  5. 搞定——Claude Code 自动就在这个文件夹里工作了

💡 找不到这个选项？ 也可以打开终端后，输入 cd （注意 cd 后面有个空格不能少），然后把文件夹从资源管理器拖进终端窗口，文件夹路径会自动填入，然后再按回车就行。cd 的意思是”进入某个文件夹”，是 Change Directory 的缩写。

方法二：把文件夹路径告诉 Claude（不需要记命令）

  1. 打开终端，输入 claude 回车启动
  2. 在对话框里直接输入文件夹的完整路径，比如： 或者 Mac：
  3. Claude Code 会自动跳转到那个路径

💡 怎么看文件夹的完整路径？

  • Windows：打开文件夹，点击地址栏（显示 此电脑 > 文档 > ... 的地方），你会看到类似 C:\Users\小明\文档 的文字，这就是绝对路径
  • Mac：在文件夹上右键 → 按住 Option 键 → 选择「将…拷贝为路径名称」，粘贴出来就是类似 /Users/小明/文档/工作总结
  • 简单理解：绝对路径就是**从电脑最顶层一路往下找到这个文件夹的”地图”**，以盘符（Windows）或 /（Mac）开头，中间用 \ 或 / 分隔

下面的例子虽然偏技术，但核心思路是一样的：Claude Code 帮你把想法直接变成文件——不管是代码、博客、还是日常文书。

功能一：长期记忆——它记得你是谁

每个用过 豆包、deepseek 的人都有这个痛点：每次新建对话，都要重新介绍一遍自己的背景。

小 T 之前也很烦这个,但之前他问过网页版的deepseek，它对此也没有好办法。但 Claude Code 有一个长期记忆系统，你只需一次告诉它你的信息：

Claude Code 会把信息存成结构化的记忆文件。下次新对话自动加载，不需要重复介绍。

后来小 T 更进一步，把个人信息整理成了一份 Markdown 档案，里面有：

  • 基本情况（学校、专业、年级、方向）
  • 技术栈（已掌握的、正在学的、计划学的）
  • 项目路径（代码在哪里、博客在哪里）
  • 性格和学习偏好（代码风格、输出风格）

之后在任何新项目目录下说一句 **”初始化，载入我的通用记忆”**，所有上下文就恢复了。

💡 节省了什么：以前每次新对话要花 5 分钟自我介绍，现在一句话搞定。一年下来至少省了几十个小时的重复劳动。

功能三：定制学习路线——它能帮你做决策

这是小 T 最意外的用法。

他在犹豫暑假要不要学 STM32 嵌入式。之前参加学校智能车社团的培训时学过一阵子，但中途因为”遇到问题没人帮、正反馈太少”退出了。现在有两份很好的资源摆在面前：

  • B 站博主 keysking 的 STM32 教程（有配套文档站 docs.baud-dance.com，自带例程和 FAQ）
  • 资深学长 Lain 写的《机器人工程师学习计划》（5万字，覆盖 8 个方向 + 考研/就业分析）

两份资料都很长，自己啃完要花几天。他把两边的关键页面喂给 Claude Code 去读，然后问：

Claude Code 给出了详细分析：

  • keysking 的教程配合了 CubeMX + HAL 库的开发方式（比小 T 以前死磕 Keil5 寄存器高效得多），例程覆盖了 GPIO → UART → PWM → ADC → I2C → SPI → FreeRTOS → 智能农场完整项目
  • Lain 的文档明确指出：嵌入式开发是本科生最容易切入的机器人方向，2026 年头部企业给本科嵌入式应届生的薪资已接近部分算法岗
  • 结论：暑假完全可以学完 STM32 核心外设，每个外设跑通后写一篇博客作为产出

小 T 后来在博客里写道：”之前我一个人瞎学，根本不知道方向对不对。现在有人帮我分析过，我才敢大胆往前推。”

💡 关键价值：不是替你决策，而是帮你消化大量信息再呈现给你判断。这就像有个学长帮你把两份几万字的资料读完、划好重点、标出跟你相关的内容。

功能四：帮你写博客——让学习有产出、有正反馈

自学者最缺的是正反馈。学了半天不知道自己到底掌握了没有，很容易泄气。

Claude Code 能把学到的知识直接转化成博客文章：

它会：

  1. 读你的 SeqTable.h 和 SeqTable.c，提取核心逻辑
  2. 用大白话解释每个函数做了什么、为什么这么设计
  3. 画示意图、列复杂度表格
  4. 生成带 frontmatter（标题、日期、标签、分类）的 .md 文件，直接放到你博客的 source/_posts/ 目录

小 T 每学完一个数据结构就发一篇。一个月下来博客已经有了好几篇有深度的文章。不仅学习更有成就感，博客本身也是未来求职作品集的一部分。

💡 正反馈循环：学一个模块 → 写一篇博客 → 收到点赞和评论 → 更有动力学下一个模块。Claude Code 帮你把「写博客」这个环节的效率提升了数倍。

功能五：Debug 排查——有人陪你一起看 bug

学编程最大的痛苦不是写不出代码，而是代码报错了看不懂。尤其是在配环境、编译、链接这些环节。

小 T 的 VS Code 配 C 语言多文件编译时，遇到了经典的 undefined reference to 'xxx' 错误：

Claude Code 读了 .vscode/tasks.json，立刻定位到问题："${file}" 只编译当前打开的单个文件，改成 "${fileDirname}\\*.c" 就好了。还给出了修改前后的对比表格和验证方法。

这个排查过程如果用搜索引擎，要学会描述问题、筛选结果、判断哪个答案适合自己的环境。小 T 以前就经常卡在这一步。现在 Claude Code 直接读他的配置文件，一步到位。

总结：它是什么，它不是什么

Claude Code 不是：

  • ❌ 替你写作业的工具
  • ❌ 让你不用动脑的捷径
  • ❌ 替代你学习的魔法

Claude Code 是：

  • ✅ 一个能记住你所有上下文的搭档
  • ✅ 一个能用你的代码来讲解的私人助教
  • ✅ 一个能帮你消化大量资料、辅助你做决策的参谋
  • ✅ 一个能帮你输出博客、让你持续获得正反馈的输出伙伴

小 T 有一句话总结得很好：

“以前一个人学，遇到坑就卡住，卡住就烦躁，烦躁就放弃。现在有了一个能记住一切的搭档——每次对话它都知道我上次学到哪儿了，遇到 bug 能帮我排查，学完一个模块还能帮我输出博客。正反馈多了，学习自然就坚持下来了。”
